生成预览

LLM 生成的 J2 模板，可直接编辑
gdal2tiles.j2
{# @id gdal2tiles #}
{# @name 生成地图瓦片 #}
{# @description 将栅格数据切分为 TMS/XYZ 格式的地图瓦片 #}
{# @concept "TMS" — Tile Map Service，一种地图瓦片服务协议 #}
{# @concept "XYZ" — 简化的瓦片坐标方案，z/x/y 格式 #}
{# @note 输入栅格需已投影到 Web Mercator (EPSG:3857) #}
{# @param input file_path required 输入栅格文件路径 #}
{# @param output_dir file_path optional 输出瓦片目录 #}
{# @param zoom string optional zoom 级别范围 default=5-15 #}
{# @param profile string optional 瓦片配置文件 default=mercator #}
{# @common_error "No input file" — 检查 input 路径是否正确 #}

gdal2tiles.py -z {{ zoom | default('5-15') }}
  --profile={{ profile | default('mercator') }}
  {{ input | quote }}
  {{ output_dir | default(workspace + '/output/' + input | basename | splitext | first + '_tiles') | safe_path }}

概念说明

TMS — Tile Map Service，一种地图瓦片服务协议，y 轴从南向北增长
XYZ — 简化的瓦片坐标方案，z/x/y 格式，y 轴从北向南增长（与 TMS 相反）
